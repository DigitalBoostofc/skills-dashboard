# Routes

Single-page application — no routing library. Mode is controlled by JS `setMode()`.

## Panels (toggled via JS show/hide)
- `#skills-panel` — default active. Skills grid with category filters.
- `#agents-panel` — Agents grid with squad cards + function filters.
- `#strategies-panel` — Multi-squad playbooks grid.
- `#guide-panel` — Usage guide with pattern sections.

## Tab nav (header)
- `#tab-skills` → skills mode
- `#tab-agents` → agents mode
- `#tab-strategies` → strategies mode
- `#tab-guide` → guide mode

## External data files
- `agents-data.js` — agent definitions
- `strategies-data.js` — strategy definitions
